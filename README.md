## php・laravel練習用アプリ
以下の機能を実装、Udemy動画を使用してCRUD機能を練習。
その後、追加機能を実装する。
- 認証機能（breezeを使用）
- ContactFormのCRUD機能
- modelの情報をCSV入出力できる機能
- Repository層とService層の分離

## 使用技術
- PHP8
- Laravel9
- Nginx
- MySQL@8
- Docker, docker-compose
